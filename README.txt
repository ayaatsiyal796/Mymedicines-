=====================================
CAREPLUS PHARMACY WEBSITE
=====================================

📁 FOLDER STRUCTURE:
-------------------
pharmacy-website/
│
├── index.html    (Main HTML file - open this in browser)
├── style.css     (All styles and responsive design)
├── script.js     (All JavaScript functionality)
└── README.txt    (This file)

🌐 HOW TO RUN:
-------------
1. Save all 3 files (index.html, style.css, script.js) in the same folder
2. Double-click index.html or open it in any modern web browser
3. Website will work offline (no internet required except for icons)

📱 PAGES INCLUDED:
-----------------
✅ Home Page - Complete details and services
✅ Medicines - Full medicines list with search
✅ Categories - Medicine categories with filtering
✅ About Us - Pharmacy information
✅ Contact Us - Contact details with map
✅ Blog - Health articles and tips

🔍 FEATURES:
-----------
- Search bar to find medicines by name or category
- Click on medicine to view full details (modal popup)
- Category browsing with medicine preview
- Fully responsive (works on mobile, tablet, desktop)
- Modern UI with Font Awesome icons

🛠️ TECHNOLOGIES:
---------------
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (No external libraries needed)

💊 MEDICINES DATABASE:
--------------------
Includes 8 sample medicines with prices and descriptions.
You can easily add more medicines in script.js (medicinesDB array).
